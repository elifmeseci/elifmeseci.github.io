**WHAT'S RUBY?**
Ruby is a language of careful balance. Its creator, Yukihiro “Matz” Matsumoto, blended parts of his favorite languages (Perl, Smalltalk, Eiffel, Ada, and Lisp) to form a new language that balanced functional programming with imperative programming.
Since its public release in 1995, Ruby has drawn devoted coders worldwide. In 2006, Ruby achieved mass acceptance. With active user groups formed in the world’s major cities and Ruby-related conferences filled to capacity.
Ruby is also completely free. Not only free of charge, but also free to use, copy, modify, and distribute.

You can also visit [about ruby](https://www.ruby-lang.org/en/about/) for more information.

**Why is Ruby needed?**
Object oriented language with scripting language are separate concepts. Object-oriented programming contains more detail and is more comprehensive. In order to use the advantages of object-oriented programming in the language structure of the script, the Ruby language was needed. 

**What are the differences between Ruby and Ruby on Rails?** 
Ruby on Rails is an open source web application development framework written in Ruby. Ruby is an object-oriented programming language.

**What is Rails?**
Ruby On Rails is an open source web application development framework written in Ruby. Ruby on Rails; It adopts the Model-View-Controller (MVC) approach, providing developers with a number of convenience to collect web server information query or edit information in the database, and process the page template.
